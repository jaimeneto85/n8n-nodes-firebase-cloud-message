# Task ID: 1
# Title: Setup Project Repository and Environment
# Status: pending
# Dependencies: None
# Priority: high
# Description: Initialize the project repository, set up the development environment, and configure essential dependencies.
# Details:
1. Create a new repository on GitHub named 'n8n-nodes-firebase-cloud-message'
2. Initialize the project with npm: `npm init -y`
3. Install necessary dependencies:
   - n8n-core@^0.160.0
   - n8n-workflow@^0.160.0
   - firebase-admin@^11.9.0
4. Set up TypeScript:
   - Install TypeScript: `npm install --save-dev typescript@^4.9.5`
   - Create tsconfig.json with appropriate settings
5. Set up linting and formatting:
   - Install ESLint and Prettier: `npm install --save-dev eslint@^8.40.0 prettier@^2.8.8`
   - Configure .eslintrc.js and .prettierrc
6. Create basic folder structure:
   /src
     /nodes
     /credentials
   /dist
7. Update package.json with appropriate scripts (build, lint, test)

# Test Strategy:
Manually verify the project structure and run `npm install` to ensure all dependencies are correctly installed. Check if TypeScript compilation works by running `npm run build`.
